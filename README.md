# BorsaZeka Dashboard

BorsaZeka Dashboard is a modern, high-performance financial analytics web application built with **Next.js**. It provides users with real-time insights, advanced data visualization, and an intuitive interface to track market trends and stock performance.

## Features

- **Real-time Analytics:** Monitor market movements with low-latency data updates.
- **Interactive Charts:** Beautiful and responsive data visualization for stock trends.
- **User Authentication:** Secure login and personalized dashboard experience via NextAuth.
- **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices using Tailwind CSS.
- **Agent Integration:** Specialized modules for AI-driven financial analysis (as detailed in `AGENTS.md`).

## Tech Stack

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Authentication:** [Auth.js (NextAuth)](https://authjs.dev/)
- **Icons & UI:** Lucide React & Radix UI
